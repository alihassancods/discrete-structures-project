System Documentation & Help

Variables: Single letters (p, q, r, s, t).

Operators:
AND (&)
OR (|)
NOT (! or ~)
IF THEN (->)
IFF (<-> or <=>)

Parentheses () for grouping.

Case-insensitive for operators.

Sets are entered as comma-separated values within curly braces {}. Elements can be numbers, strings (quoted or unquoted if simple), etc.

Example: {1, 2, "apple", banana}

Universal Set (U) is required for Complement operations.

Quantifiers: FOR ALL, EXISTS.

Variables: Single letters (x, y, z).

Predicates:
Basic: EVEN(var), ODD(var), PRIME(var), DIVISIBLE_BY(var1, var2).
Comparison: >(var1, var2), <(var1, var2), =(var1, var2), >=(var1, var2), <=(var1, var2).
Logical operators are same as Propositional Logic.

Domain is entered as a Set.

Set A is entered as a standard Set.

Relation R is entered as a set of ordered pairs within curly braces {}. Each pair is (element1, element2).

Example: {(a,b), (b,c), (c,a)}
